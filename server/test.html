      Job Description

      Fortinet is looking for passionate and talented problem solvers to join our main DevOps team for strengthening the security of internal environment which is a key part of Fortinets software R&D.

      The successful candidate will be responsible for testing the existing services and infrastructure while designing and developing feature enhancements.

      Job Responsibilities
      • Supporting end users and testing existing applications and infrastructure.
      • Implementing test automation environment.
      • Provisioning and testing Linux machines with security hardened image to enhance the security of internal environment.
      • Analyzing and resolving problems associated with the Linux operating system, hardware, firmware and application.
      • Coordinating and supporting internal users for machine migration.
      • Implementing and improving Linux security hardening automation.

      Job Skills Required
      • Experience with python development & scripting.
      • Experience with Linux, bare-metal and virtual machines.
      • Understanding of file system, storage configurations and data replication.
      • Experience with Linux shell scripting.
      • Good knowledge of SVN, GIT and best practice in release management.
      • Team player with strong interpersonal, written and verbal communication skills.
      • Outstanding troubleshooting and debugging skills.
      • Ability to multi-task and quickly react to system/service issues.

      Nice To Haves
      • Knowledge of TCP/IP networking, firewalls, switching, and routing.
      • Familiarity with web services (JSON, REST, etc.)
      • Experience with provisioning Linux using Metal-As-A-Service (MASS).

      Educational Requirements
      • Bachelor's Degree in Computer Science, Electrical Engineering or related discipline
